# Campus Survival Simulator (C++)

## Overview
Campus Survival Simulator is a console-based application designed to help students manage their academic life, improve productivity, and enhance campus safety awareness.

## Features
- Study planner and timetable management  
- Pomodoro focus timer  
- Daily goals system with save/load functionality  
- GPA/CGPA calculator with career suggestions  
- Skill gap analysis  
- Campus safety reporting system  
- Threat level indicator and safety tips  

## Technologies Used
- C++  
- File handling (for saving reports and goals)  
- Windows-specific libraries (windows.h, conio.h)  

## How to Run
1. Open the project in a C++ IDE such as CodeBlocks or Dev C++  
2. Compile the source file  
3. Run the executable  

## Notes
- This project is designed for Windows systems due to the use of Windows-specific libraries.
